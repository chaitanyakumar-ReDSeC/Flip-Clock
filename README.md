## ✨ Key Features

* **Massive 24-Hour Display**: A bold, large-scale flip-clock interface designed to fill the screen for maximum legibility from a distance.
* **Dual-Mode Functionality**: Seamlessly toggle between a real-time **Digital Clock** and a high-precision **Countdown Timer**.
* **Screen Wake Lock**: Integrated **Stay Awake** technology prevents your device from dimming or sleeping while the clock is active—perfect for use as a dedicated desk accessory.
* **Immersive Fullscreen**: Double-tap (or double-click) anywhere on the background to enter or exit immersive fullscreen mode.
* **Emergency Visual Cues**: The timer interface transitions to a high-visibility pulsing red alert during the final 10 seconds and upon completion.
* **Adaptive Themes**: Switch between a deep "Onyx Dark" and "Cloud Light" mode to match your environment.

## 🛠️ Built With

* **React 19** & **TypeScript**
* **Motion (Framer)** for fluid flip-card animations
* **Tailwind CSS 4.0** for a sleek, responsive UI
* **Lucide React** for minimalist iconography
* **Screen Wake Lock API** for persistent display utility

## 🚀 Deployment

This application is optimized for hosting on **GitHub Pages**.

> **Note on Stay Awake**: The Screen Wake Lock feature requires a secure context (HTTPS). When hosted on GitHub Pages, the "Always-On" display functionality is fully enabled by default.

## 🖱️ Interaction Guide

| Action | Result |
| :--- | :--- |
| **Double Click** | Toggle Fullscreen Mode |
| **Top Right Icon** | Toggle between Clock and Timer |
| **Bottom Right Icon** | Toggle Dark/Light Mode |
| **Timer Inputs** | Click numbers to set custom duration |
